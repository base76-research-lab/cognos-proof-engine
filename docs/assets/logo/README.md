# CognOS Logo Assets

## Files

- `cognos-logo-horizontal.svg` — primary logo for hero/header sections.
- `cognos-logo-mark.svg` — square icon for favicon/app icon/social profile.

## Recommended usage

- Landing page hero: use `cognos-logo-horizontal.svg` at 320–560px width.
- GitHub/social avatar: use `cognos-logo-mark.svg`.
- Keep clear space around logo (at least the size of the center circle radius).
- Prefer dark backgrounds for best contrast.

## Color intent

- Deep navy background = infrastructure/stability.
- Teal/blue gradient = trust + intelligence.
- Node-ring motif = verification network / signal engine.
