Quick Start (3 Steps)

1. Clone

git clone https://github.com/base76-research-lab/operational-cognos.git cd operational-cognos

2. Install & Run

pip install -r requirements.txt export COGNOS_MOCK_UPSTREAM=true python3 -m uvicorn --app-dir src main:app --port 8788

3. Test

curl -X POST http://127.0.0.1:8788/v1/chat/completions \ -H 'Content-Type: application/json' \ -d '{ "model": "gpt-4o-mini", "messages": [{"role":"user","content":"Is this safe?"}], "cognos": {"mode":"monitor"} }'

Or use Docker

docker-compose up # In another terminal: curl -X POST http://127.0.0.1:8788/v1/chat/completions ...

Or install Python SDK

pip install cognos-sdk from cognos import CognosClient client = CognosClient() response = client.chat( model="gpt-4o-mini", messages=[{"role":"user","content":"What's 2+2?"}], mode="monitor" ) print(f"Decision: {response.cognos.decision}")
